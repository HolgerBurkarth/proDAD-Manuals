# This is an overview of the user manuals for proDAD GmbH products.

ProDAD GmbH is a German software company that specializes in video effects and production software.
Since its founding in 1990, the company has developed creative tools and solutions for video creators, filmmakers, and content producers.

Integrating ProDAD's products with popular video editing platforms benefits both professional and amateur users in the digital media creation industry.
